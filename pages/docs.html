Langages de Programmation

Python - Un langage de programmation simple et puissant utilisé pour l'automatisation, avec des bibliothèques comme `Selenium` et `PyAutoGUI`.
Java - Utilisé avec des frameworks comme `JUnit` et `TestNG` pour les tests automatisés.
JavaScript - Langage clé pour l'automatisation côté front-end, avec des outils comme `Cypress` et `Puppeteer`.
Ruby - Langage apprécié pour les tests automatisés avec `Capybara` et `RSpec`.
PHP - Langage de script côté serveur, souvent utilisé pour le développement web.
Go - Langage de programmation moderne, connu pour sa simplicité et sa performance.
Swift - Langage de programmation pour le développement d'applications iOS et macOS.
Kotlin - Langage moderne pour le développement Android, compatible avec Java.
Rust - Langage de programmation axé sur la sécurité et la performance.
Scala - Langage qui combine la programmation orientée objet et la programmation fonctionnelle.
SQL - Langage de requête pour gérer et manipuler des bases de données.

Frameworks et Outils de Test

Selenium - Framework pour l'automatisation des tests de navigateurs web.
Robot Framework - Framework d'automatisation des tests basé sur des mots-clés.
Serenity BDD - Framework de test pour Java qui facilite la création de tests automatisés.
Jest - Framework de test JavaScript, souvent utilisé avec React.
Mocha - Framework de test JavaScript pour Node.js.
TestNG - Framework de test pour Java, inspiré de JUnit.
Jasmine - Framework de test pour JavaScript.
Cucumber - Outil de test basé sur le comportement (BDD) qui utilise Gherkin pour écrire des scénarios.
qTest - Outil de gestion des tests et de suivi des défauts.
HP ALM - Outil de gestion du cycle de vie des applications.

CI/CD et Outils d'Intégration Continue

Jenkins - Outil d'intégration continue largement utilisé pour automatiser les builds et les tests.
Travis CI - Service d'intégration continue pour GitHub.
CircleCI - Outil d'intégration continue et de déploiement continu.
GitLab CI - Outil d'intégration continue intégré à GitLab.
Azure DevOps - Outil de gestion de projet et d'intégration continue de Microsoft.

Outils de Versionning

GitHub - Plateforme de versionning très utilisée, avec des fonctionnalités pour l'intégration continue.
GitLab - Outil de gestion de version et de collaboration.
Bitbucket - Outil de gestion de version et de collaboration, souvent utilisé avec Jira.
Sourcetree - Client Git graphique pour Windows et Mac.

Outils Divers

Docker - Outil de virtualisation des applications permettant d'automatiser la création, le test et le déploiement d'environnements.
Kubernetes - Système d'orchestration de conteneurs pour automatiser le déploiement, la mise à l'échelle et la gestion des applications conteneurisées.
Ansible - Outil d'automatisation pour la gestion de la configuration et le déploiement d'applications.
Terraform - Outil d'infrastructure as code pour gérer les ressources cloud.
Postman - Outil pour tester et documenter les API.
Jira - Outil de gestion de projet et de suivi des problèmes.
Kibana - Outil de visualisation de données pour Elasticsearch.
PuTTY - Client SSH et telnet pour Windows.
TIBCO - Outils d'intégration et de gestion des données.
Dynatrace - Outil de surveillance des performances des applications.
DBeaver - Outil de gestion de base de données universel.
